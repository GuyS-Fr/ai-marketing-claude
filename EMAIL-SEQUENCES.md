**1. "Quelle tâche est-ce que je refais plus de 5 fois par semaine de manière identique ?"**

Les relances clients à J+3, les rappels de maintenance, les suivis de devis non-signés... Si ça dépend de votre mémoire, un workflow le fait mieux que vous. Sans oubli. Sans exception.

Si vous faites vos devis le soir et votre facturation le dimanche, c'est un signal d'alarme. Ce temps-là devrait être automatisé pour que vous le récupériez.

| Tâche | Avant | Après automatisation |

| Devis client | 48h, saisie manuelle | 2 min, formulaire → devis auto |

| Facturation | 2h chaque vendredi | Automatique à la signature du devis |
| Réponse client hors horaires | 0 (personne ne répond) | Chatbot IA 24/7 |

## Implementation Notes

### ESP recommandé
Pour une micro-entreprise solo :

| Outil | Prix | Adapté pour | Recommandation |
|-------|------|------------|----------------|
| **Brevo (ex-Sendinblue)** | Gratuit jusqu'à 300 emails/jour | Welcome + Nurture | **Recommandé** — français, RGPD-compliant, automatisations incluses |
| **ConvertKit** | $15/mois | Créateurs et consultants | Bon alternative si audience internationale |
| **Lemlist** | $59/mois | Cold outreach | Pour la séquence Cold Outreach uniquement |

### Setup minimal

1. **Semaine 1 :** Créer le PDF lead magnet + formulaire Brevo + séquence Welcome (7 emails)
2. **Semaine 2 :** Ajouter le formulaire sur le site (homepage, blog, popup exit-intent) + configurer la séquence Nurture
3. **Semaine 3 :** Construire la base de prospects locaux + configurer la séquence Cold Outreach
4. **En continu :** Mesurer, A/B tester, itérer

### Intégration avec le site

- Le formulaire d'inscription doit être **sur chaque page** (sidebar ou inline)
- Un **popup exit-intent** doit apparaître quand le visiteur s'apprête à quitter le site
- Chaque **article de blog** doit avoir un formulaire en fin d'article
- Le formulaire doit **promettre la checklist** + **gérer les attentes** ("Vous recevrez aussi un email par semaine avec des méthodes d'automatisation pour PME")
