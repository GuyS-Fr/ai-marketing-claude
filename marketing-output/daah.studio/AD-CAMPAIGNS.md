## 1. STRATÉGIE PUBLICITAIRE GLOBALE

### Positionnement publicitaire DAAH Studio

DAAH Studio occupe un positionnement hybride rare : une agence digitale à taille humaine, ancrée localement en Drôme/Ardèche, mais avec des offres scalables nationalement (abonnements). Le message publicitaire s'articule autour de trois axes :

1. **La proximité et la confiance** — Une vraie personne derrière le projet, pas une agence anonyme
2. **La flexibilité sans engagement** — Des formules d'abonnement modernes qui cassent les codes du "site one-shot"
3. **L'accessibilité assumée** — Des prix clairs dès la première annonce, pour les TPE/PME qui ont peur des devis flous

### Architecture du funnel publicitaire

```
TOFU — Notoriété & Découverte
│  Objectif : faire connaître DAAH Studio
│  Canaux : Meta (Reach/Video), Google Display
│  Message : "Un site pro + SEO + mises à jour à 99€/mois"
│  KPI : CPM < 5€, portée maximale locale
│
MOFU — Considération & Intérêt
│  Objectif : visites qualifiées sur daah.studio
│  Canaux : Meta (Traffic), Google Search local
│  Message : "Ton site vitrine clé en main — Valence, Drôme, Ardèche"
│  KPI : CPC < 1,50€, taux de rebond < 55%
│
BOFU — Conversion & Contact
   Objectif : demande de devis / appel découverte
   Canaux : Google Search (intent fort), Retargeting Meta
   Message : "Prenons 30 min pour parler de ton projet"
   KPI : CPA < 45€, taux de conversion formulaire > 4%
```

### Segmentation des audiences prioritaires

| Segment | Profil | Canal prioritaire | Message clé |
|---------|--------|------------------|-------------|
| Artisan / commerçant local | 30–55 ans, Drôme/Ardèche, pas encore en ligne | Google Local + Meta | "Un site pro livré en 3 semaines" |
| Solopreneur / consultant | 28–45 ans, freelance, France entière | Meta + LinkedIn | "Design à la carte, 0 engagement" |
| Dirigeant TPE/PME | 35–55 ans, 1–20 salariés, Auvergne-Rhône-Alpes | Google + LinkedIn | "DAAH Premium : tout inclus, 99€/mois" |
| Créateur d'activité | 25–40 ans, lancement d'entreprise | Meta + Google | "Ton identité visuelle + site dès 800€" |
