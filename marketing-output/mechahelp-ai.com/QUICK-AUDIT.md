## Top 3 Quick Wins — Priorites immediates

### #1 — Ajouter 5 temoignages clients
```
Effort    : Faible (2-3 jours)
Impact    : +15 a +25% sur le taux de conversion
Action    : Collecter 5 retours clients existants, les afficher en homepage
            Format : photo + prenom + secteur + resultat concret
```

### #2 — Corriger le canonical brise + generer un sitemap XML
```
Effort    : Moyen (1 semaine dev)
Impact    : Passage de 0 a N pages indexees en 2-4 semaines
Action    : Implementer SSR ou pre-rendu statique
            Soumettre sitemap.xml dans Google Search Console
            Verifier balise canonical sur chaque page
```

### #3 — Reduire a 2 CTA + ajouter une FAQ
```
Effort    : Faible (1-2 jours)
Impact    : +8 a +12% sur le taux de clic
Action    : Identifier les 2 actions cles (ex. "Trouver un expert" / "Rejoindre la communaute")
            Ajouter FAQ repondant aux objections : prix, difference vs Malt, garanties
```
