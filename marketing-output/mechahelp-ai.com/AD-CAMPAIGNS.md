## 1. STRATÉGIE PUBLICITAIRE GLOBALE

### Positionnement publicitaire

MechaHelp est la première marketplace francophone d'accompagnements en IA & automatisation à la session. Le message publicitaire s'articule autour de trois leviers :

1. **La peur de prendre du retard** — Les concurrents automatisent, pas encore vous
2. **La promesse de résultat rapide** — Un expert, une session, un livrable opérationnel
3. **L'accessibilité et le sans-risque** — À partir de 50€, garantie satisfaction, sans engagement

### Architecture du funnel publicitaire

```
TOFU — Notoriété & Acquisition
│  Objectif : faire découvrir MechaHelp
│  Canaux : Meta (Video/Reach), TikTok Ads
│  Message : "Automatisez avec des experts IA francophones"
│  KPI : CPM < 8€, taux de vue vidéo > 30%
│
MOFU — Considération & Inscription
│  Objectif : inscription gratuite sur la plateforme
│  Canaux : Meta (Traffic/Lead), Google Search
│  Message : "24 accompagnements disponibles — inscription gratuite"
│  KPI : CPC < 1,20€, CPL < 8€
│
BOFU — Conversion & Réservation
   Objectif : réservation d'une session payante
   Canaux : Meta Retargeting, Google Search (intent fort)
   Message : "Réservez votre session — Expert disponible cette semaine"
   KPI : CPA < 35€, ROAS > 4
```

### Segmentation de l'audience

| Segment | Profil | Canal prioritaire | Message |
|---------|--------|-----------------|---------|
| Freelances | 25–45 ans, auto-entrepreneur, secteur digital | Meta + TikTok | Gain de temps, prospection |
| Dirigeants PME | 35–55 ans, 5–50 salariés, B2B | LinkedIn + Google | ROI, délégation, process |
| Entrepreneurs e-com | 25–45 ans, boutique Shopify/WooCommerce | Meta + Google | SAV auto, CRM, prospection |
| Créateurs de contenu | 20–40 ans, YouTuber/TikToker/créateur | TikTok + Instagram | Contenu IA, productivité |

## 7. TESTS A/B RECOMMANDÉS

### Tests Meta Ads — Mois 1

| Variable | Version A | Version B | Critère de victoire |
|----------|----------|----------|-------------------|
| Accroche visuelle | Texte sur fond uni | Photo/vidéo témoignage | CPL le plus bas |
| CTA | "En savoir plus" | "Réserver maintenant" | CTR le plus élevé |
| Angle | Douleur (Angle 1) | Preuve sociale (Angle 2) | CPA le plus bas |
| Audience | Intérêts IA/Auto | Lookalike inscrits existants | Volume + CPL |
| Format | Image statique | Vidéo 15s | CPM + CTR |

### Tests Google Ads — Mois 1

| Variable | Test |
|----------|------|
| Titre 1 | "Expert IA à la session" vs "Automatisez votre Business" |
| Description | Angle ROI vs Angle social proof |
| URL de destination | Homepage vs page service spécifique |
| Type de correspondance | Exact match vs Phrase match |

### Tests LinkedIn — Mois 1–2

| Variable | Test |
|----------|------|
| Format | Single Image vs Document Ad |
| Audience | Dirigeants PME 10–50 sal. vs Freelances |
| Message | ROI et économie vs Différenciation vs consultants |
| Lead Gen Form | 3 champs vs 5 champs |

### Règles de test A/B

1. Ne tester qu'une variable à la fois
2. Budget minimum par version : 100€ ou 1 000 impressions avant analyse
3. Critère de victoire statistique : 95% de confiance (outil : AB Testguide.com)
4. Archiver les perdants — ne pas les relancer sans modifier un élément
5. Documenter chaque test dans un tableau de bord partagé (Notion recommandé)
